# Anime TUI 🌸

Консольний додаток для пошуку та перегляду аніме прямо у вашому терміналі!
Швидкий, зручний, без реклами та зайвого візуального сміття.

Підтримувані провайдери:
- **АніЛібрія** (Anilibria)
- **YummyAnime**
- **HDRezka**

## Особливості 🌟
- 🚀 **Блискавичний пошук** із живим оновленням результатів (powered by `fzf`).
- 🔄 **Швидке перемикання** між провайдерами гарячими клавішами (`Alt+1`, `Alt+2`, `Alt+3`).
- 🏷️ **Фільтрація за жанрами** через зручне меню `Alt+G` або ручний ввід (наприклад: `genre:Романтика`).
- 🖼️ **Прев'ю обкладинок** тайтлів прямо в терміналі!
- ❤️ **Улюблені тайтли**: зберігайте свої улюблені аніме та переглядайте їх через спеціальне меню (`Alt+4`).
- 📺 Вбудований стрімінг за допомогою `mpv`.
- 📝 Підтримка відображення рейтингів, описів та коментарів для тайтлів.
- 🇺🇦 Локалізований інтерфейс.

## Вимоги 🛠️
Для роботи програми необхідні наступні залежності:
- **Python 3.8+**
- **fzf** (для зручного TUI інтерфейсу)
- **mpv** (для відтворення відео)
- **yt-dlp** (для отримання прямих посилань на відео)
- Python бібліотеки: `requests`, `climage`, `HdRezkaApi`

## Автоматичне встановлення (Linux / macOS) ⚡
Найпростіший спосіб встановити Anime TUI зі всіма залежностями та додати команду в систему — використати наш інсталятор:

```bash
chmod +x install.sh
./install.sh
```

Цей скрипт автоматично:
1. Перевірить та встановить відсутні системні пакети (`fzf`, `mpv`).
2. Створить віртуальне середовище Python в `~/.local/share/anime-tui`.
3. Встановить необхідні Python бібліотеки.
4. Додасть команду `anime-tui` до `~/.local/bin/`.

**Важливо:** Переконайтеся, що `~/.local/bin` є у вашій змінній середовища `$PATH`. 
Якщо ні, додайте цей рядок у ваш `~/.zshrc` або `~/.bashrc`:
```bash
export PATH="$HOME/.local/bin:$PATH"
```

## Використання 🎮
Після встановлення просто відкрийте термінал і введіть:
```bash
anime-tui
```

### Гарячі клавіші:
- Введіть назву для пошуку
- Використовуйте `↑` та `↓` для навігації
- `Enter` — обрати тайтл (далі обрати серію і якість)
- `Alt+1` — Шукати на АніЛібрія
- `Alt+2` — Шукати на YummyAnime
- `Alt+3` — Шукати на HDRezka
- `Alt+4` — Відкрити "Улюблені"
- `Alt+G` — Відкрити меню вибору жанрів
- `Esc` — Вихід
